# ❄️ SnowRunner DLC Unlocker: Automatic DLC Activator & Unlocker Tool

**SnowRunner DLC Manager** is a lightweight all-in-one utility designed to automatically unlock and activate downloadable content for **Snowrunner** on Steam.  
This tool streamlines DLC management for supported expansions, content packs, and additional modules.

> ⚠️ A legitimate copy of **SnowRunner** (Steam / Epic Games / Console) is required to use this tool.

# ⚡ Installation & Setup Guide

## 1️⃣ Download the Tool

Download the latest release from the Releases page.

## 2️⃣ Extract Files

Extract the archive into a folder of your choice.

## 3️⃣ Run the Application

Launch the executable normally (Administrator optional depending on setup).

## 4️⃣ Select Game Installation

Point the tool to your SnowRunner installation directory.

## 5️⃣ Scan DLCs

Click **Scan DLCs** to detect installed and owned content.

## 6️⃣ Launch Game

Use the built-in launcher to start SnowRunner with verified content.

# 🔧 Troubleshooting

| Issue | Possible Fix |
|------|--------------|
| DLC not detected | Verify game files via Steam/Epic launcher |
| Missing expansions | Ensure DLC is installed in platform library |
| Tool not launching | Install latest .NET runtime |
| Game launches without DLC | Re-check installation directory |
| Scan fails | Restart tool with administrator permissions |

# 💡 Tips & Notes

* Always keep your game updated to the latest version
* Verify game files if DLC does not appear
* Re-run scan after installing new expansions
